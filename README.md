# 💰 **Financy**

## Sobre o Projeto

O Financy é uma aplicação FullStack para organização de finanças por meio da gestão de transações e categorias. Segue abaixo o passo a passo para executar da aplicação:

1. Clone o repositório:
   ```bash
   git clone https://github.com/basilioarth/Financy.git
   ```

2. Siga a documentação do backend:

    🗄️ [Back-end](https://github.com/basilioarth/Financy/tree/main/backend#readme)

3. Siga a documentação do frontend:

    🖥️ [Front-end](https://github.com/basilioarth/Financy/tree/main/frontend#readme)
